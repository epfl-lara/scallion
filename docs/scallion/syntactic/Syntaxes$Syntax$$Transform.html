Transform 

case class Transform[A, B](function: (A) ⇒ B, inverse: (B) ⇒ Seq[A], inner: Syntax[A]) extends Syntax[B] with Unary[A] with Product with Serializable

Syntax that applies a function on the parsed values of the inner syntax and an inverse function on the printed values given to the inner syntax.

function

The function to apply on produced values.

inverse

The function to apply on printed values.

inner

The inner syntax.
